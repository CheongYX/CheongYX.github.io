操作系统四大特性，用王者荣耀一看就懂

操作系统的四大特性，用《王者荣耀》和电脑日常来聊聊

学操作系统的时候，经常会冒出几个让人头疼的词：并发、并行、共享、异步。
这些概念看上去抽象，但其实和我们玩游戏、用电脑时的体验差不多。
我试着用《王者荣耀》加上电脑日常场景，来聊聊这四个特性。

1. 并发性（Concurrency）

打团的时候，你看到五个英雄在同一秒同时开技能，火花四溅。
但实际上，一个 CPU 并不能真的让他们“同时”放招，而是在不同英雄之间快速切换指令，速度快到让你误以为大家一起动。

在电脑上，就是那种：
你一边开着 Chrome 播放 YouTube 音乐，一边写 Word 文档，微信消息还在不停弹。
从你角度看，这三个任务都在“同时跑”，但实际上 CPU 是在它们之间来回切换。
切得够快，就像团战时的“看似同时”。

2. 并行性（Parallelism）

并行更好理解：就是真·同时。
你和四个朋友五排开黑，每人一台手机。大家各自的操作各自处理，互不干扰。

在电脑上，这就像多核 CPU：
一个核心在拼命渲染你的视频，另一个核心在流畅运行你的游戏。
打开任务管理器，你能看到好几个 CPU 核心同时在“卷”。这就是并行。

3. 共享性（Sharing）

共享就是：资源大家都能用，但要讲规矩。
红BUFF、蓝BUFF，谁都想要，但一个时间点只能一个人拿（互斥共享）。
而塔的视野，大家能一起享有（同时共享）。

在电脑上，你打开两个软件都想用打印机。
打印机一次只能给一个任务打印，另一个只能乖乖排队。
但像系统里的一个只读文件，很多程序可以同时打开，这就像“全队共享视野”。

4. 异步性（Asynchronism）

异步就是：事情不是一口气完成的，而是走走停停，由系统来调度。
比如后羿的大招，一箭飞出去，你就能去平A、走位，不用干等箭飞到敌人脸上。
等箭命中，游戏再提示你（眩晕了），你再接后续操作。

在电脑上，这就是：
你点开一个视频，播放器立刻能播前几秒，剩下的系统还在后台加载。
又或者你开始下载文件，进度条在转，但你完全可以一边上网冲浪一边等。

🎯 小结

操作系统看似抽象的四大特性，其实就在我们日常体验里：

  • 并发：一个 CPU 快速切换，像团战中假装大家一起放招。
  • 并行：多核 CPU 真·同时执行，就像五个人各自拿手机开黑。
  • 共享：资源大家能用，但有的要排队，有的能一起享。
  • 异步：不用干等，系统会在时机合适的时候通知你接着干。

下次再有人问你“并发和并行有什么区别”，你就甩他一句：
👉 并发是假装大家一起打，并行是真开黑！🎮
